Datatypes

In js Datatype can be 

* Object
* Primitive types like (number, string etc)

**Number** : in js it's represents both decimal and floating point numbers.


Variable declarations
*********************

**let** : used in most declarations, the value assigned here is mutable
**const** : declare constants , which means value assigned is immutable.

Note : when using const we always need a initial value eg : "const state"
will give you a exception : VM655:1 Uncaught SyntaxError: Missing initializer in const declaration
